\section{Conclusiones}

El proyecto permitió construir un sistema de clasificación para estimar hospitalización por dengue con base en síntomas y variables complementarias.

La comparación sistemática entre modelos clásicos y ensambles permitió seleccionar el mejor enfoque según \texttt{F1-macro} en test.

Como trabajo futuro se recomienda:
\begin{itemize}
    \item ampliar validación temporal por periodos epidemiológicos,
    \item incluir calibración de probabilidades para soporte clínico,
    \item monitorear deriva de datos en operación.
\end{itemize}
